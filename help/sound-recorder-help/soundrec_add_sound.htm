To record a sound into a sound file

  1. Make sure you have an  connected to your computer.
  2. On the File menu, click Open, and then double-click the sound file you want to modify.
  3. Move the slider to the place in the file where you want to record sound.
  4. To begin recording, click .
  5. On the File menu, click Save As.

  • If you add a sound to the middle of an existing sound file, the new sound replaces the original sound after the insertion point.
